class zoop_wrapper.models.invoice.BillingConfiguration(allow_empty=False, **kwargs)¶

Bases: zoop_wrapper.models.base.ZoopObject

Represents a billing configuration object.

It has dynamic types!

Can be a Fee or a Discount.

And can be Fixed or Percentage.

So it can be Fee and Fixed. Or Fee and Percentage. Or Discount and Fixed. Or Discount and Percentage.

Percentage can be just PERCENTAGE_MODE or DAILY_PERCENTAGE_MODE or MONTHLY_PERCENTAGE_MODE.

Fixed is FIXED_MODE.

The type identifier is the mode attribute value as in the Zoop API.

The attr is_discount identify if it’s a Discount or not.

value representing if it’s a Fee or Discount type

Type

percentage¶

Type
